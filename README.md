## 🦾 기능
1. **목표 설정** <br>
  목표 시간과 성공 또는 실패 시 듣고 싶은 말을 설정할 수 있습니다.<br>

2. **목표 수행** <br>
   설정한 목표 시간대로 원형 타이머가 작동됩니다.<br>
   성공 또는 실패 시 설정한 문장이 나타납니다.<br>

3. **성공 쿠키 저장** <br>
   목표 달성 횟수만큼 쿠키가 저장됩니다.<br>
